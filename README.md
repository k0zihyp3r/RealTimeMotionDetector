# RealTimeMotionDetector

A lightweight, high-performance C++ application utilizing the OpenCV 4 library for real-time motion detection. The project processes video streams, isolates moving objects from static backgrounds using adaptive Gaussian Mixture Models (MOG2), cleans background noise using morphological operations, and highlights detected entities with dynamic bounding boxes.

## Image Processing Pipeline

The program processes each video frame through a multi-stage computer vision pipeline:
1. **Grayscale Conversion:** Reduces data complexity from 3 channels (BGR) to 1 channel.
2. **Gaussian Blur:** Smooths out camera digital noise.
3. **MOG2 Subtractor:** Evaluates pixel movement relative to a learned background model and extracts moving masks.
4. **Binary Thresholding:** Discards shadows and segments high-confidence motion areas.
5. **Erosion:** Disintegrates tiny pixel clusters generated by environmental disturbances.
6. **Morphological Closing:** Fills voids inside the vehicles to prevent multiple boxes on a single car.
7. **Contour Extraction:** Computes geometric shapes and wraps them inside `cv::Rect` bounding boxes.

## Prerequisites

To compile and run this project, you need:
* C++ Compiler supporting C++11 or higher (GCC, MSVC, Clang).
* OpenCV 4.x installed and properly configured in your system paths/IDE environment.
* A test video file placed under the relative directory: `../res/test.mp4`.
